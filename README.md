# api-gerar-escala
Api SLIM para gerar escala mensal

# Utilização
1 - Clonar projeto em repositório local

2 - Rodar comando:
  `docker-compose up`
  
3 - rodar comando:
  `docker exec -it  api-gerar-escala-php  bash`
  
4 - rodar comando:
  `composer install`

5 - rodar comando:
  `php -S localhost:8000`
